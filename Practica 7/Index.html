¿Qué es HTML?

Es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.

¿Qué es CSS?

La hoja de estilo en cascada es un lenguaje usado para definir la presentación de un documento estructurado escrito en HTML y derivados.

¿Qué es JavaScript?

Es un lenguaje de secuencias de comandos para la web. Es un lenguaje interpretado, lo que significa que no necesita un compilador para traducir su código como C o C ++.
